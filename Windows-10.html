Windows 10

Screenshot of Microsoft Windows 10

  • Windows 10 is a major release of Microsoft's Windows NT operating system. It is the direct successor to Windows 8.1, which was released nearly two years earlier.

  • Microsoft initially aimed to have Windows 10 installed on over one billion devices within three years of its release

Features of Windows 10

  • The Start Menu was reintroduced in Windows 10, combining the traditional Start Menu with live tiles from Windows 8.

  • Microsoft's virtual assistant, Cortana, is integrated into Windows 10, allowing users to perform voice commands and search using natural language.

  • Windows 10 introduced Microsoft Edge, a new web browser meant to replace Internet Explorer.

  • Users can create and manage multiple desktops to organize open applications and windows, making multitasking more efficient.

  • A biometric authentication system that allows users to log in using facial recognition, fingerprint, or iris scanning.

  • Designed for pen-enabled devices, Windows Ink provides features like a virtual ruler, sticky notes, and sketchpad for digital drawing.

  • The Microsoft Store provides a platform for users to download and install apps, games, and other content.

  • The Action Center consolidates notifications and provides quick access to settings like Wi-Fi, airplane mode, and screen brightness.
