Windows 98

Screenshot of Microsoft Windows 98

  • Windows 98 is a consumer-oriented operating system developed by Microsoft as part of its Windows 9x family of Microsoft Windows operating systems.

  • it is the successor to Windows 95, and was released to manufacturing on May 15, 1998, and generally to retail on June 25, 1998.

  • Like its predecessor, it is a hybrid 16-bit and 32-bit[3] monolithic product with the boot stage based on MS-DOS.

Features of Windows 98

  • Windows 98 featured a more polished and user-friendly interface compared to its predecessor, Windows 95.

  • USB support was improved, making it more convenient to connect and use USB peripherals.

Advantages of Windows 98

  1. Windows 98 included built-in support for DVD drives, allowing users to play DVD movies without the need for third-party software.
