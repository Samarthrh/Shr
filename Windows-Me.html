Windows Me

Screenshot of Microsoft Windows Me

  • It is the successor to Windows 98, and was released to manufacturing on June 19, 2000, and then to retail on September 14, 2000.

Features of Windows Me

  • Windows Me was targeted specifically at home PC users, and included Internet Explorer 5.5, Windows Media Player 7 and the new Windows Movie Maker software, which provided basic video editing and was designed to be easy to use for consumers.

  • The Help and Support Center was introduced, offering a centralized location for assistance with troubleshooting and accessing documentation.
