Windows 8.1

Screenshot of Microsoft Windows 8.1

  • Windows 8.1 was released on August 27, 2013.

Features of Windows 8.1

  • Users can customize the Start screen with various tile sizes and rearrange them to suit their preferences.

  • The lock screen gained the ability to display a slideshow of images from the user's picture library.

  • Windows 8.1 introduced several security enhancements, including Windows Defender improvements and the introduction of the Secure Boot feature.

  • Windows 8.1 included Internet Explorer 11 with improved performance, support for more web standards, and enhanced touch capabilities.

  • The Windows Store was redesigned to provide a better user experience, with improved app discovery and updates.

  • Windows 8.1 included built-in support for 3D printers, allowing users to print 3D objects directly from their devices.
